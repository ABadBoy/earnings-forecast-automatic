# A-Share Earnings Forecast Auto-Publisher

This is an automated system that fetches daily A-Share earnings forecast announcements, uses a Large Language Model (LLM) to read the raw PDF documents, extracts growing companies, and automatically generates and publishes a formatted article to your WeChat Official Account drafts.

## Features

1. **Automated Data Fetching**: Automatically searches and downloads the latest earnings forecast announcements (PDFs) from CNINFO (巨潮资讯) for a given date.
2. **LLM Native Extraction**: Instead of fragile rule-based parsing, the system passes the raw text of the announcements directly to an LLM (e.g., DeepSeek).
3. **Smart Generation**: The LLM acts as a financial analyst, filtering out only the companies with earnings growth, analyzing their data, and natively generating a professional, ready-to-publish article.
4. **WeChat Integration**: The generated Markdown is converted to styled HTML with inline CSS and automatically pushed to your WeChat Official Account's Draft box, using your preset cover image.

## Quick Start

### 1. Installation

Install the required dependencies:

```powershell
python -m pip install -r requirements.txt
```

### 2. Configuration

Copy the `.env.example` file to a new file named `.env` and fill in your actual credentials:

```env
# LLM Settings
LLM_API_KEY=your_llm_api_key
LLM_BASE_URL=https://api.deepseek.com
LLM_MODEL=deepseek-chat

# WeChat Settings
WECHAT_APP_ID=your_wechat_app_id
WECHAT_APP_SECRET=your_wechat_app_secret
# Get this using `python list_materials.py`
WECHAT_THUMB_MEDIA_ID=your_cover_image_media_id
```

### 3. Run the Pipeline

To run the entire pipeline (Fetch -> Parse -> LLM Write -> WeChat Publish) for a specific date:

```powershell
python -m earnings_forecast run-cninfo --date 2026-07-01 --use-llm --publish
```

**Arguments:**
- `--date`: The date of the announcements to fetch (e.g., `2026-07-01`).
- `--use-llm`: Enables the LLM pipeline instead of static templates.
- `--publish`: Automatically publishes the result to the WeChat Draft box.
- `--max-pages`: (Optional) The maximum number of CNINFO pages to crawl.
- `--limit`: (Optional) The maximum number of PDFs to process.

### 4. Utilities

To list the image materials currently uploaded to your WeChat Official Account (to find a valid `WECHAT_THUMB_MEDIA_ID` for your covers):

```powershell
python list_materials.py
```

## Generated Artifacts

During the run, the system will save the intermediate files locally:

```text
data/downloads/cninfo/YYYY-MM-DD/*.pdf      # Raw downloaded PDFs
data/outputs/YYYY-MM-DD/announcements.json  # Raw extracted text from PDFs
data/outputs/YYYY-MM-DD/article.md          # Markdown article generated by LLM
data/outputs/YYYY-MM-DD/article.html        # Styled HTML ready for WeChat
```
